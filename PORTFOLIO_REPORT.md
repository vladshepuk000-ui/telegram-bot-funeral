# Звіт по завданням менторського списку

## 1. Перенести сайт на Vercel ✅

- Створено репозиторій `vladshepuk000-ui/portfolio` на GitHub
- Задеплоєно на Vercel: `portfolio-six-taupe-97.vercel.app`
- Підключено автодеплой — кожен `git push` оновлює сайт автоматично

**Баги:**
- `npx vercel` не запускався через заблоковані скрипти в PowerShell → вирішено через `Set-ExecutionPolicy`
- Токен Vercel був недійсний → зроблено `vercel login` заново
- Після першого деплою статус "No Production Deployment" → вирішено через `npx vercel --prod`
