# Vape Shop — Розумна вітрина + автомаркетинг

Система автоматизації онлайн-магазину рідин та картриджів для вейпів.

## Документація
- [Опис проекту](docs/project.md) — що це, навіщо, структура
- [Roadmap](docs/roadmap.md) — етапи розробки з задачами
- [Архітектура](docs/architecture.md) — схема системи, БД, FSM, деплой

## Швидкий старт

```bash
# Клонувати та встановити залежності
pip install -r requirements.txt

# Скопіювати та заповнити .env
cp .env.example .env

# Запустити бота
python bot/main.py
```

## Етапи
1. Telegram-бот з каталогом і FSM замовленням
2. База клієнтів + авторозсилки
3. Планувальник задач (APScheduler)
4. Автопостинг у групи регіону (Telethon)
5. Веб-дашборд (FastAPI)
