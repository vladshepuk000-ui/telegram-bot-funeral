# Проект: Vape Shop Bot — Розумна автоматизація магазину

## Суть проекту
Повна автоматизація невеликого вейп-магазину через Telegram.
Мало товарів — не проблема: система сама стежить за асортиментом, приймає замовлення,
відповідає клієнтам і залучає нових покупців з сусідніх міст.

**Учбовий проект** — але з реальною практичною цінністю.
Після завершення можна запустити як справжній магазин.

---

## Проблеми які вирішує
| Проблема | Рішення |
|---|---|
| Постійно одні й ті ж питання | Бот відповідає автоматично |
| Ручне оновлення асортименту | Пишеш пост у канал → каталог оновлюється сам |
| Мало клієнтів з інших міст | Автопостинг у регіональні групи |
| Немає статистики | Веб-дашборд з графіками |
| Забуваєш написати клієнтам | Авторозсилки та нагадування |

---

## Технічний стек — все безкоштовно

| Компонент | Технологія | Чому |
|---|---|---|
| Telegram-бот | aiogram 3.x + FSM | Головний інтерфейс |
| База даних | SQLite → PostgreSQL | SQLite безплатно назавжди |
| Планувальник | APScheduler | Вбудований в Python |
| Парсер каналу | Telethon | Безкоштовний user API |
| Автопостинг | Telethon | Той самий пакет |
| AI-асистент | Google Gemini API | Безкоштовний tier: необмежено для навчання |
| Веб-дашборд | FastAPI + Jinja2 | Безкоштовний фреймворк |
| Веб-сайт | Vue.js | Безкоштовно |
| Хостинг бота | Railway або Render | Безкоштовний план |
| Хостинг PWA | Vercel або Netlify | Безкоштовно назавжди |

> Google Gemini API — безкоштовний tier дає 1500 запитів/день.
> Для невеликого магазину цього більш ніж достатньо.
