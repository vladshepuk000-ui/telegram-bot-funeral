# Roadmap — Етапи розробки

## Етап 1 — Telegram-бот з каталогом і замовленням ✅ ГОТОВО
**Мета:** базовий бот який замінює ручні переписки з клієнтами

### Зроблено:
- [x] Структура папок, .env, requirements.txt, .gitignore
- [x] Ініціалізація SQLite бази даних
- [x] Таблиці: products, customers, orders, order_items, waitlist
- [x] /start — привітання, головне меню, збереження клієнта в БД
- [x] Каталог з фото, категорії: рідини / картриджі / системи
- [x] FSM замовлення: товар → кількість → телефон → доставка → адреса → коментар → оплата → підтвердження
- [x] Вибір доставки: Нова Пошта або Самовивіз
- [x] Вибір оплати: картка (скріншот), накладений платіж, готівка
- [x] Скріншот оплати — клієнт надсилає фото, адмін отримує
- [x] Залишок зменшується після скріншоту (картка) або одразу (НП/готівка)
- [x] Скасування замовлення повертає залишок
- [x] Кнопка скасування на кожному кроці замовлення
- [x] Перевірка залишку при виборі кількості
- [x] /cancel — скидає FSM у будь-який момент
- [x] /stop — клієнт відписується від розсилок
- [x] Сповіщення адміну після нового замовлення
- [x] FAQ — автовідповіді на часті питання
- [x] Контакт — юзернейм і телефон продавця
- [x] Адмін: /addproduct, /editproduct, /removeproduct, /orders, /setstatus, /restock
- [x] Кнопка "Повторити останнє замовлення"
- [x] Кнопка "Мої замовлення" — статуси замовлень клієнта
- [x] Waitlist — "Сповісти коли з'явиться" + автосповіщення при поповненні

**Результат:** клієнт сам переглядає товари і оформлює замовлення без твоєї участі ✅

## Етап 2 — Розсилки + планувальник задач ✅ ГОТОВО
**Мета:** система сама нагадує клієнтам і слідкує за магазином

### Зроблено:
- [x] APScheduler підключений до бота
- [x] /broadcast — ручна розсилка всім підписаним клієнтам з превью
- [x] /addtemplate — додати шаблон для авторозсилки
- [x] /templates — переглянути всі шаблони
- [x] Авторозсилка щопонеділка о 12:00 (чергує шаблони)
- [x] Нагадування клієнтам через 21 день після останнього замовлення
- [x] Щотижневий звіт адміну: замовлення, сума, топ товар, нові клієнти
- [x] Запит відгуку клієнту після доставки (/setstatus done → ⭐ кнопки)
- [x] /reviews — адмін переглядає всі відгуки із середньою оцінкою

**Результат:** клієнти повертаються самі, ти отримуєш тижневий звіт і відгуки автоматично ✅
